# ⚙️ W++ Token Analyzer

A modern, web-based **Lexical Analyzer** built for the **CS-310 Compiler Construction** course. This application processes source code written in the custom **W++** language, breaking it down into categorized tokens and providing detailed statistical insights through an interactive dashboard.

---

## 🚀 Features

- **🔍 Live Tokenization:** Instantly breaks code into Keywords, Identifiers, Literals, Operators, and Separators.
- **📊 Statistical Reports:** Generates an overview of token frequency, unique types, and average tokens per line.
- **📈 Line Distribution:** Visualizes code density across different lines.
- **💎 Identifier & Literal Tracking:** Lists all unique identifiers and constants used, including their frequency and line numbers.
- **🌊 Token Stream:** A visual "stream" view where you can hover over tokens to see their specific types.
- **📂 File Support:** Upload `.wpp` or `.txt` files directly into the analyzer.

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Syne & JetBrains Mono Fonts).
- **Logic:** JavaScript (Vanilla JS) for the Regex-based Tokenizer and data processing.
- **Styling:** Modern dark-themed UI with glassmorphism and grid overlays.

---

## 💻 How to Run

1. Clone the repository to your local machine.
2. Open `index.html` in any modern web browser (Chrome, Edge, or Firefox).
3. Type your **W++** code in the editor or click the **Sample** button to load a test program.
4. Click **▶ Analyze** to generate the statistical report.
